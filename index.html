Dicas de como aprender a programar usando HTML

      Introdução

      O HTML (HyperText Marked Language) é a linguagem de marcação padrão para criação de páginas da Web.

      Existem diversas versões do HTML que foram criadas conforme a linguagem foi evoluindo até chegar na versão dos dias atuais mais usada que é o HTML5.

      O HTML descreve a estrutura de uma página da Web e ele consiste em uma série de elementos que informam ao navegador como exibir o conteúdo criado para a página.

      O que são Elementos

      Um elemento HTML é tudo, desde a tag start até a tag end. Um elemento HTML é definido por uma marca de início, algum conteúdo e uma tag final. Os elementos podem ser aninhados, ou seja esses elementos podem conter outros elementos.Temos como exemplos quatro elementos HTML: html body h1 p.

      O elemento html é o elemento raiz e define todo o documento do HTML. Ele tem uma tag de início e uma tag de fim. Então, dentro do elemento há um elemento: html body.

      O que são Atributos

      Os atributos HTML fornecem informações adicionais sobre elementos HTML.Todos os elementos HTML podem ter atributos e os atributos são sempre na tag de início. Os atributos geralmente vêm em pares nome/valor como: nome="valor".

      A tag href define um hiperlink e o atributo href, especifica a URL da página para onde é direcionado o link.

      A tag src é usada para incoporar uma imagem em uma página HTML e o atributo src especifica o caminho para a imagem a ser exibida.

      O que são Textos

      Um parágrafo sempre começa em uma nova linha e é geralmente um bloco de texto . Um parágrafo sempre começa em uma nova linha, e os navegadores adicionam automaticamente algum espaço em branco(uma margem) antes e depois de um parágrafo. O elemento HTML define um parágrafo.

      O que são Links

      Os links são encontrados em quase todas as páginas da Web.Os links permitem que os usuários cliquem em seu caminho de página para página.

      Links HTML são hiperlinks. Você pode clicar em um link e ir para outro documento. Quando você mover o mouse sobre um link, a seta do seu mouse se transformará em uma mãozinha.

      *Nota: O link não precisa ser um texto, um link pode ser uma imagem ou qualquer outro elemento HTML ! O atributo mais importante do elemento é o atributo href, que indica o destino do link (a). Por padrão, os links aparecerão da seguinte forma em todos os navegadores:

        Um link não visitado é sublinhado e azul.

        Um link visitado é sublinhado e roxo.

        Um link ativo é sublinhado e vermelho.
